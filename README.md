# Task-1: Data Immersion & Wrangling

This repository contains my Task-1 submission for the ApexPlanet Data Analytics Internship.

## Objective
To understand a raw dataset, identify data quality issues, and prepare a clean, analysis-ready dataset.

## Dataset
- Raw data provided for analysis

## Work Performed
- Created a data dictionary to describe each column
- Identified missing values and duplicate records
- Cleaned and transformed data using Python (Pandas)
- Standardized date formats and corrected inconsistencies
- Generated a final cleaned dataset

## Files Included
- raw_data.csv – Original dataset
- cleaned_data.csv – Cleaned dataset
- data_dictionary.xlsx – Data dictionary
- data_cleaning.py – Python cleaning script

## Tools Used
- Python
- Pandas

## Outcome
A clean dataset ready for further analysis and visualization.
